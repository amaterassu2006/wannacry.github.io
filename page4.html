Les failles de Windows

La propagation de WannaCry a été permise par l'exploit zero-day EternalBlue. Développé par la NSA dans le cadre de son programme pour stocker et exploiter des failles de cybersécurité, EternalBlue a été volé par le groupe de hackers Shadow Brokers, qui a diffusé l'exploit le 14 avril 2017. EternalBlue cible les systèmes d'exploitation Windows. Petya est un autre ransomware ayant utilisé le code EternalBlue.
