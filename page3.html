Le fonctionnement de WannaCry

Le ransomware WannaCry se propageait via l'exploit EternalBlue, développé par l'Agence de sécurité nationale (NSA) américaine. Cet exploit cible une vulnérabilité dans le protocole Server Message Block (SMB) dans les versions obsolètes de Microsoft Windows, telles que Windows XP, Windows 8 et Windows Server 2003. WannaCry installait une backdoor pour permettre un accès facilité au système par le hacker, puis chiffrait les données pour demander une rançon. Les victimes étaient invitées à payer sous trois jours, après quoi la rançon était doublée, et les données pouvaient être supprimées une semaine plus tard si le paiement n'était pas effectué.
